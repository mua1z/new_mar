# Merawan Mohammed - Leadership Portfolio

A high-end, modern, animated, and 3D-enhanced personal leadership website for Merawan Mohammed, a Woreda Administration Officer in Ethiopia.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Features

- **Multi-page Layout**: Home, Profile, Education, Experience, Initiatives, and Contact.
- **Modern UI**: Clean, professional administrative aesthetic with "gov-blue" and "gov-emerald" accents.
- **3D Effects**: Interactive mouse-tracking parallax and 3D tilt effects on cards.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Multilingual**: Support for English (EN), Afaan Oromo (OM), and Amharic (AM).

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
